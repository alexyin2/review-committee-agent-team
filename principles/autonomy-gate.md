# Autonomy Gate(自主閘判準)

> 用「**可逆性 × 影響範圍**」決定:自己做,還是停下問人。受治理、版本化、走 PR。

## 自動執行(可逆 且 影響小)

- 讀提交、跑各維度分析。
- 寫 findings、更新 case README、貼**草稿建議 / brief** 到審查頻道。
- 請專案團隊補缺漏欄位。

## 停下問人(不可逆 或 影響大)→ 走 PR + 指定人 merge

- 發出真正 gate 住上線的 **go/no-go 裁決**。
- **否決**一個專案。
- 任何專案團隊會當成「官方決定」的東西。

## 配套

- 共享頻道持續以**摘要級**顯示 agent 在做什麼,讓人能在走到關鍵點**之前**就介入(有感)。
- 規則本身被治理:改這份要開 PR。
